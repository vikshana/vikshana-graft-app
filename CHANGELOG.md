# 0.1.0 (2025-12-23)

### Features and enhancements

- **Chat:** Natural language conversation interface for interacting with Grafana
- **Chat:** Toggle between Standard (fast) and Deep Research (complex reasoning) modes
- **Chat:** Support for image and text file attachments in conversations
- **Chat:** Markdown rendering with syntax highlighting and Mermaid diagram support
- **Chat:** Expandable tool execution visualization with results
- **LLM:** Integration with Anthropic, OpenAI-compatible APIs, Ollama, and LM Studio via Grafana LLM plugin
- **MCP:** Grafana Model Context Protocol tools for querying dashboards, datasources, and more
- **History:** Browse, search, and continue previous conversations
- **Prompts:** Save and reuse commonly used prompts with customizable templates
- **Context:** Automatic injection of current user, dashboard, and datasource context
- **Backend:** Go backend with health monitoring and OpenTelemetry observability
